# QuantumVault Protocol (Tabi L1)

Utility token and base infrastructure for the QuantumVault ecosystem. Matches the official .pharos domain.

## Overview
QuantumVault is an ERC20 implementation designed for secure asset management and reward distribution within the Tabi L1 network.

## Setup
```bash
npm install
npx hardhat compile
```

## Contracts
- `QuantumVault.sol`: Core vault token with owner-governed minting.
